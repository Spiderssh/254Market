# Kenya Financial Intelligence Dashboard

A **live**, real-time financial dashboard for Kenyan markets hosted on **GitHub Pages** with automated data updates via GitHub Actions.

## ✨ Features

- **Real-time forex rates** (USD/KES, EUR/KES) - updates every 60 seconds
- **Live crypto/commodity prices** via CoinGecko API
- **Auto-updated NSE data** every 30 minutes during market hours
- **Live TV streams** (Citizen, NTV, KTN, K24)
- **60-second countdown refresh** with manual refresh button
- **Client-side predictive analytics** using trend calculations
- **Risk meter** (1-10 scale) with color coding
- **Breaking news** with severity alerts

*Built with 🇰🇪 for Kenya — March 2026*
